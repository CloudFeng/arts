# ARTS项目说明

ARTS 是[左耳朵耗子](https://github.com/haoel)发起的一个活动，具体如下：

- Algorithm：每周至少做一个leetcode的算法题;
- Review：阅读并点评至少一篇英文技术文章；
- Tip：学习至少一个技术技巧；
- Share：分享一篇有观点和思考的技术文章；

**你需要坚持至少一年!**
